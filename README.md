# Your-News (v2)

학교 공지 알림 서비스 서버입니다.

## ✨ Features
### 권한별
 - 관리자 (진행 예정)
     - 사용자 관리
     - 소식 관리
     - 게시글 관리
 - 사용자
     - 소식 구독 (받고자 하는 소식 구독)
     - 게시글 작성

## 🚀 Tech Stacks
 - Language: Java
 - Build tool: Gradle
 - Frameworks & Libraries: Spring Boot, Spring Data JPA, Spring Security, Lombok, JJWT, Redis, firebase
 - Databases & Caching: MySQL, Redis
 - DevOps & Cloud: GitHub Actions, Docker, AWS (예정)
